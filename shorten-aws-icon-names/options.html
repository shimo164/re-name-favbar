Shorten AWS Service Names Mappings

To add a mapping, write it in the format "original name, new name".
Each mapping should be on a new line.

Example: DynamoDB, DDB
This would map "DynamoDB" to "DDB".

Example: DynamoDB,
This would map "DynamoDB" to empty.

By default, the suffixes "AWS" and "Amazon" are removed from service names.
However, mappings will override this behavior.
Example: Amazon Timestream, Amazon Timestream
This would keep "Amazon Timestream" as it is.
